## Overview

AllControl is a smart stock management system designed to help businesses track inventory, analyze stock flow, and generate predictive requisitions. It combines clean design with powerful automation, making stock management more efficient and error-free.

## Key Features

- **Inventory Tracking**: Manage incoming and outgoing stock with a clear dashboard and real-time updates.  
- **Predictive Requisition**: Integrated algorithms to forecast stock needs and reduce shortages.  
- **Detailed Analytics**: Visualized statistics and reports to support decision-making and optimize operations.  
- **User-Friendly Interface**: Built with a responsive and modern UI for both desktop and mobile.  

## Technologies Used

- **Next.js**: For building a fast, responsive, and modern front-end.  
- **Elysia.js + Prisma + PostgreSQL**: For the backend API and robust database management.  
- **Tailwind CSS**: For a clean, consistent, and customizable design.  
- **Chart.js**: For generating interactive statistics and reports.  

## Challenges and Learnings

One challenge was building a flexible system that could adapt to different business needs while keeping the interface simple. Solving this required modular database design and reusable UI components. It also improved my experience with predictive modeling and API design.  

## Outcome

AllControl is now a scalable stock management solution, reducing inventory errors and saving businesses time. It demonstrates how practical automation and modern design can simplify complex operations and deliver measurable impact.
